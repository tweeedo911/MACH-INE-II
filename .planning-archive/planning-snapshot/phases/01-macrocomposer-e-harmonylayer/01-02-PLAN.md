**`applyVoiceLeading(prevChord, candidateNotes, maxLeap)` — funzione interna:**
```javascript
function applyVoiceLeading(prevChord, candidateNotes, maxLeap) {
  if (!prevChord || prevChord.length === 0) return candidateNotes.slice();
  // Per ogni voce nel prevChord, trova la nota candidata piu' vicina entro maxLeap
  const used = new Set();
  return prevChord.map(prev => {
    let best = candidateNotes[0];
    let bestDist = Infinity;
    for (const n of candidateNotes) {
      if (used.has(n)) continue;
      const dist = Math.abs(n - prev);
      if (dist <= maxLeap && dist < bestDist) {
        bestDist = dist;
        best = n;
      }
    }
    // Se nessuna candidata entro maxLeap, cerca la piu' vicina senza vincolo
    if (bestDist === Infinity) {
      for (const n of candidateNotes) {
        if (used.has(n)) continue;
        const dist = Math.abs(n - prev);
        if (dist < bestDist) {
          bestDist = dist;
          best = n;
        }
      }
    }
    used.add(best);
    return best;
  });
}
```
NOTA: se prevChord ha piu' voci di candidateNotes, le voci in eccesso mantengono la nota precedente (natural sustain). Se candidateNotes ha piu' voci, le extra vengono ignorate.
