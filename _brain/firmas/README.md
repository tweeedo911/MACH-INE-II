# Firmas — gesti narrativi

> Le firme sono interventi del regista (`firma.js`) che alterano lo stato globale per creare archi drammatici.

## Inventario

- **gelo** — congelamento densità, tensione massima
- **convergenza** — tutte le track convergono a stessa key/note
- **vuoto** — silenzio strategico, palco aperto
- **densityCap** — tetto dinamico per evitare saturazione
